# MVP仕様

* ユーザー登録 -> done

* ログイン／ログアウト -> done

* 質問投稿 -> done

* 質問回答 -> done

* 質問一覧（解決済み／未解決それぞれ） -> done
  * 日付順（新しい質問が上）
  * ページングは無し

* 質問投稿者が質問を解決済みにできる -> done

* 質問詳細画面 -> done
  * 回答一覧
    * 日付順（新しい回答が上）

* マイページ -> done
  * 質問一覧
  * 回答一覧
